AppName: TheDementorsNetwork
AppType: Android App
Verified on Android 5.0.2 API 21

This app is somehow similar to the maurders map which we had seen in the famous harry potter movies.
At the first time launch of the app it will ask you to enter your name so that it can register you
with the online databse. After registration it will generate a userid corresponding to that phone.
And next time you open the map it will directll take you to the map showing all the people registred
with this app. It shows realtime position of all the users. Make sure you grant location access to the
app while using it, if you don't the app will simply close and will not show anything.
